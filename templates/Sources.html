ChatBot:

ChatBot is a virtual machine that has ability of replying to any question raised. A Chatbot is generally implemented on a Data Set which is Cornell movie dialog corpus in this case. the Data Set is so trained that each word is converted into a token and some set of words converts into neuron which are capable of answering the input question. The neurons selects words depending on weights trained seperately for each data.
